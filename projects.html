Сервис онлайн тестирования

Проект Quizful предназначен для онлайн-тестирования специалистов в сфере информационных технологий. Quizful является community-driven сервисом и развивается во многом благодаря усилиям сообщества ИТ-специалистов.

На Quizful вы найдете более 140 тестов:

  • тесты по программированию (C++, Java, C#, 1C, PHP, Python, Ruby)
  • тесты по базам данных (SQL, Oracle, MS SQL, MySql)
  • тесты по веб-технологиям (HTML, CSS, JavaScript, HTTP, jQuery)
  • тесты по управлению проектами

Цель проекта - предоставить качественный сервис, который бы не только проверял уровень имеющихся знаний, но и помогал в получении новых знаний.
